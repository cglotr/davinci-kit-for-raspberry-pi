Code herunterladen
======================

Alle in diesem Kit verwendeten Beispielprogramme sind in unserem offiziellen GitHub-Repository gespeichert.  
Verwenden Sie den folgenden Befehl, um das vollständige Projekt auf Ihren Raspberry Pi herunterzuladen.

Repository klonen
--------------------

#. Melden Sie sich bei Ihrem Raspberry Pi an und führen Sie Folgendes aus:

   .. raw:: html
   
       <run></run>
   
   .. code-block:: bash
   
      cd ~/
      git clone https://github.com/sunfounder/davinci-kit-for-raspberry-pi.git --depth 1

#. Wechseln Sie in das Projektverzeichnis:

   .. raw:: html
   
       <run></run>
   
   .. code-block:: bash
   
      cd ~/davinci-kit-for-raspberry-pi/

#. Listen Sie die Dateien auf:

   .. raw:: html
   
       <run></run>
   
   .. code-block:: bash
   
      ls

#. Sie sehen eine Struktur ähnlich der folgenden:

   .. code-block:: text
   
      davinci-kit-for-raspberry-pi/
      ├── c/
      ├── datasheet
      ├── iot/
      ├── nodejs/
      ├── python-pi5/
      ├── python/
      ├── scratch/
      └── README.md

Übersicht über die Projektstruktur
----------------------------------

Hier ist eine kurze Einführung in jeden Ordner:

* **c/**  

  Beispiele und Bibliotheken in der Programmiersprache C für Benutzer, die lieber in C auf dem Raspberry Pi programmieren.

* **datasheet/**  

  Datenblätter, Spezifikationen und Referenzdokumente für die im Projekt verwendeten Hardwarekomponenten.

* **iot/**  

  IoT-bezogene Beispiele, einschließlich der Anbindung an die Blynk-Plattform, Sensordemonstrationen und Kommunikationsmodule.

* **nodejs/**

  Node.js-Beispiele für Benutzer, die JavaScript-basierte Projekte auf dem Raspberry Pi entwickeln.

* **python-pi5/**  

  Python-Beispiele, die mit der ``GPIO Zero``-Bibliothek geschrieben wurden und speziell für den **Raspberry Pi 5** optimiert sind.

* **python/**  

  Python-Beispielprogramme, die mit der ``RPi.GPIO``-Bibliothek geschrieben wurden und für die meisten Raspberry-Pi-Boards geeignet sind.

* **scratch/** 

  Scratch-Programmierbeispiele, die für Anfänger entwickelt wurden, um grafische Programmierung zu erlernen.

* **README.md**  

  Grundlegende Informationen über das Repository und allgemeine Anweisungen.

Sie können nun den Ordner auswählen, der Ihrer bevorzugten Programmiersprache oder Ihrem Projekttyp entspricht, und mit dem Ausführen der Beispiele beginnen.
