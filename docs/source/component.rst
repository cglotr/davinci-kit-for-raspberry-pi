Komponentenvorstellung
================================================

Nach dem Öffnen der Verpackung überprüfen Sie bitte, ob alle Komponenten mit der Produktbeschreibung übereinstimmen, und stellen Sie sicher, dass sich alle Teile in einwandfreiem Zustand befinden.

.. image:: img/component_list.jpg
    :width: 90%

Dieser Abschnitt bietet einen Überblick über jede im Kit enthaltene Komponente.  
Zu jedem Element finden Sie:

* Eine kurze Einführung  
* Das Funktionsprinzip  
* Die Projekte, in denen die Komponente verwendet wird
