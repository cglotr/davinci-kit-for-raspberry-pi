Introducción de Componentes
================================================

Después de abrir el paquete, comprueba si todos los componentes coinciden con la descripción del producto y asegúrate de que cada pieza esté en buen estado.

Esta sección ofrece una visión general de cada componente incluido en el kit.

* Una breve introducción  
* El principio de funcionamiento  
* Los proyectos en los que se utiliza el componente
