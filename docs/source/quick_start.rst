===============================

Cette section a pour objectif de vous apprendre à installer Raspberry Pi OS, à configurer le Wi-Fi sur le Raspberry Pi et à accéder à distance au Raspberry Pi afin d’exécuter le code correspondant.

Si vous êtes déjà familiarisé avec le Raspberry Pi et que vous pouvez ouvrir la ligne de commande sans difficulté, vous pouvez alors passer les trois premières parties et compléter directement la dernière partie.
